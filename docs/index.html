Get Started in 5 Minutes

Only a Gemini API key is needed. Get one free at Google AI Studio.

1

Clone & Configure

Clone the repo and add your API key to local.properties.
git clone https://github.com/zero2005x/RokidAIAssistant.git
cd RokidAIAssistant
cp local.properties.template local.properties
# Edit local.properties → Add GEMINI_API_KEY
2

Build & Install

Build both apps and install them on your devices.
./gradlew :phone-app:installDebug    # Phone
./gradlew :glasses-app:installDebug # Rokid Glasses
3

Connect & Talk

Put on your glasses, pair via Bluetooth, and start talking to your personal AI assistant!
